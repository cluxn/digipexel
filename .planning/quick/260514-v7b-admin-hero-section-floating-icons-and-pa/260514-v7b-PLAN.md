Create `backend/api/upload.php` — a PHP endpoint that accepts `multipart/form-data` POST with a field named `file`, validates it, stores it, and returns the public URL.

Implementation requirements:
- `require_once '../common.php';` at top for `send_json_headers()` and `json_resp()`
- Call `send_json_headers()` immediately
- Only accept POST; reject others with `json_resp('error', null, 'Method not allowed')`
- Validate `$_FILES['file']` exists and has no upload error (`UPLOAD_ERR_OK`)
- Validate MIME type is one of: `image/png`, `image/jpeg`, `image/gif`, `image/svg+xml`, `image/webp`. Read from `$_FILES['file']['type']` — also validate via `mime_content_type()` to prevent spoofing. Reject non-images with `json_resp('error', null, 'Invalid file type')`
- Enforce max size: 2MB (`$_FILES['file']['size'] > 2 * 1024 * 1024`). Reject with `json_resp('error', null, 'File too large (max 2MB)')`
- Sanitize filename: `preg_replace('/[^a-zA-Z0-9._-]/', '_', basename($_FILES['file']['name']))`, prepend a unix timestamp: `time() . '_' . $safeName`
- Upload directory: `__DIR__ . '/../../uploads/'` (this resolves to `backend/uploads/` relative to project root). Create with `mkdir(..., 0755, true)` if it doesn't exist.
- Move file: `move_uploaded_file($_FILES['file']['tmp_name'], $uploadDir . $finalName)`
- Determine the public URL base from the request: `$proto = (!empty($_SERVER['HTTPS']) && $_SERVER['HTTPS'] !== 'off') ? 'https' : 'http'; $host = $_SERVER['HTTP_HOST']; $urlBase = $proto . '://' . $host;`
- Construct public URL: `$urlBase . '/backend/uploads/' . $finalName`
- Return: `json_resp('success', ['url' => $publicUrl], 'File uploaded')`
- Wrap entire logic in try/catch, return `json_resp('error', null, $e->getMessage())` on exception

Also create the uploads directory placeholder: `backend/uploads/.gitkeep` (empty file so the directory is committed but files inside are ignored — add `backend/uploads/*` and `!backend/uploads/.gitkeep` to `.gitignore` if not already present).

4. Add `handleIconImageUpload` function (alongside `updateIconSlot`, before the return):
```tsx
const handleIconImageUpload = async (slotIndex: number, e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (!file) return;
  const formData = new FormData();
  formData.append("file", file);
  try {
    const res = await fetch(`${API_BASE_URL}/upload.php`, { method: "POST", body: formData });
    const json = await res.json();
    if (json.status === "success" && json.data?.url) {
      updateIconSlot(slotIndex, "imageUrl", json.data.url);
    } else {
      alert("Upload failed: " + (json.message ?? "Unknown error"));
    }
  } catch {
    alert("Upload failed. Check backend connection.");
  }
  // Reset input so same file can be re-selected
  e.target.value = "";
};
```
